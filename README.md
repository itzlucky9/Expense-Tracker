# 💰 Expense Tracker App

A simple and powerful Expense Tracker app built using Flutter to manage daily expenses and track spending.

---

## 🚀 Features
- Add expenses
- Delete expenses
- Track daily spending
- Simple and clean UI

---

## 🛠️ Tech Used
- Flutter
- Dart
- (Hive / Sqflite - jo tu use kar raha hai)
